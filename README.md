Tehnologi web laboratorul 2.

1. Fiecare obiect din items din fisierul JSON este un resource.
2. http://localhost:5500/data.json#items/0
-http - Protocolul folosit de browser pentru a accesa resursa.
-localhost - Serverul local pe care rulează aplicația.
-5500 - portul serverului.
-/data.json calea catre fisierul JSON
-#items/0 - primul element din lista items
3. #items/0 - este dinamic, restul sunt statice.
4. Aplicatia este document‑centric cu un strat minim de interactivitate tehnică.
